🐍 Solution 3: Create Conda Environment (Recommended)
System Python kadang punya masalah dengan package dependencies. Lebih baik pakai conda environment:
cmd# Install conda dulu jika belum ada (download Miniconda/Anaconda)
# Atau coba dengan venv dulu:

# Buat virtual environment
python -m venv tf-directml-env

# Activate environment
tf-directml-env\Scripts\activate

# Install packages
pip install --upgrade pip
pip install tensorflow-directml
